Escape From Dungeon

Escape From Dungeon is a Third Person melee combat game for PC which was one of my early projects that I made using Unreal Engine C++ to learn the core concepts of Gameplay ,UI, AI and various programming concepts such as interfaces, strcuts, polymorphism inheritance and multithreading used in Unreal Engine.

Roles and Responsibilities

  • Implemented game mechanics and systems in Unreal Engine 4 using C++, including character creation, animations, and user input.
  • Developed a fully functional melee combat game with features such as enemy behavior, item management, and health/stamina bars.
  • Designed interactive menus, including a Pause menu, and implemented game data saving/loading functionality.
  • Created immersive gameplay elements like floating platforms, timers, trigger volumes, and hazards.
  • Utilized C++ concepts for game development, expanding skills and knowledge in Unreal Engine.

bool Snake::WallCollision()
{
	// if the snake touches the bounds
	if (snakePosition.Y < 1 || snakePosition.Y > SCREEN_HEIGHT - 2) 
	{
		return true;
	}

	// if the snake collides with the side walls then spawn from the opposite wall
	if (snakePosition.X > SCREEN_WIDTH - 2)          
	{
		snakePosition.X = 0;
		return false;
	}

	// if the snake collides with the side walls then spawn from the opposite wall
	if (snakePosition.X < 1)                            
	{
		snakePosition.X = SCREEN_WIDTH - 2;
		return false;
	}

	for (int i = 0; i < snakeLength - 1; i++)
	{
		// if the snake collides with its own body part
		if (snakePosition.X == body[i].X && snakePosition.Y == body[i].Y)                       
		{
			return true;
		}
	}
	return false;
}
